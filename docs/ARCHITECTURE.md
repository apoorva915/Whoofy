# Architecture Documentation

## System Overview

The Whoofy AI Reel Verification System is built with Next.js and TypeScript.

## Components

### Services Layer
- Video processing and analysis
- Transcription services
- Detection services (objects, brands, text, sentiment)
- Verification services (eligibility, authenticity, brand integration)

### External Integrations
- Instagram Graph API
- Apify Scraper
- Shazam API
- NoteGPT API

### API Routes
- Verification endpoint
- Campaign management
- Creator management
- Submission management
